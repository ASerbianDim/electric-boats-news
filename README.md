# Electric Boats News

A blog about electric boats, marine batteries, and maritime electrification.

## Deploy to Netlify (Free)

1. Create a GitHub repository
2. Push this folder to the repo
3. Go to [Netlify](https://app.netlify.com)
4. "Add new site" → "Import an existing project"
5. Select your GitHub repo
6. Done! Your site is live with free SSL

## Custom Domain

After deployment, in Netlify:
- Domain settings → Add custom domain
- Supports free SSL automatically

## Content Management

To add new articles, create `.html` files in the root folder following this template:

```html
---
title: Your Article Title
date: 2026-03-04
category: news
---

<article>
  <h2>Your Title</h2>
  <p>Your content here...</p>
</article>
```

## Monetization

- Google AdSense: Replace `ca-pub-XXXXXXXXXX` with your AdSense ID
- Amazon Associates: Replace affiliate links with your IDs
- Newsletter: Connect to ConvertKit or Mailchimp
